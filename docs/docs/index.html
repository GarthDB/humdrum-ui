Typography

Description

Basic responsive typography.

There are two typography breakpoints at 400px and 740px view-width;

There are two expected font families

  • serif: 'Roboto Slab', serif
  • sans-serif: 'Source Sans Pro', sans-serif

Add the Google Font include <link> tag to the <head> in your project. 

<link href="https://fonts.googleapis.com/css?family=Roboto+Slab:400,700|Source+Sans+Pro:400,700" rel="stylesheet">

Grid

Description

A simple 6-column, responsive grid.

To use the grid, use the .hd-grid class on the container.

<div class="hd-grid"></div>

Columns are defined using column classes:

  • .hd-grid__1col
  • .hd-grid__2col
  • .hd-grid__3col
  • .hd-grid__4col
  • .hd-grid__5col
  • .hd-grid__6col

The grid uses two breakpoints by default. It will switch to a single column grid under 400px, and larger height above 740px view-width.

Responsive Breakpoints

Description

Typography and grid components are responsive that change depending on the browser width using CSS Media Queries.

There are three breakpoints to use in your custom components, but only two breakpoints in use in the default components.

  • small: 400px
  • medium: 740px
  • large: 1080px
