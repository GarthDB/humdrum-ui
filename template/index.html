Getting Started

Welcome to HumdrumUI! Let's get into it.

Installation

The simplest way to start:

  1. Download the latest version
  2. Grab the index.css from the dist directory
  3. Include it in your project.
  4. Add the Google Font include <link> tag to the <head> in your project. 
    <link href="https://fonts.googleapis.com/css?family=Roboto+Slab:400,700|Source+Sans+Pro:400,700" rel="stylesheet">

NPM Installation (Advanced)

Alternatively you can use npm to install HumdrumUI into your project.

In your project, install humdrum-ui

$ npm install --save humdrum-ui

Once it's installed you can use the Sass source fils in your project using something like Webpack. The sass-loader can be configured to bring in node package files with a config like this:

{
  loader: 'sass-loader',
  options: {
    "includePaths": [
      require('path').resolve(__dirname, 'node_modules')
    ]
  }
}
